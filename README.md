# Prompter

A professional teleprompter for macOS with real-time speech-to-text auto-scrolling.

## Features

- **🎤 Speech Tracking** — Automatically scrolls as you speak using Apple Speech recognition
- **⚡ Real-time Alignment** — Intelligent matching keeps the highlighted word in sync with your voice
- **🎨 Beautiful Themes** — Dark and light modes with customizable fonts and styling
- **📐 Flexible Layout** — Adjustable window size, position, and text alignment
- **🖱️ Hover Controls** — Pause and control playback with mouse hover
- **📊 Progress Bar** — Visual indicator of your position in the script
- **⌨️ Keyboard Shortcuts** — Quick access to play/pause and settings

## Requirements

- macOS 14.0+
- Apple Silicon or Intel Mac
- Microphone access for speech tracking

## Installation

Download the latest release from the [Releases](https://github.com/imran/prompter/releases) page.

## Usage

1. **Paste your script** into the text editor in Settings
2. **Enable Speech Tracking** from the Speech settings tab
3. **Click Play** or press Space to start
4. **Start speaking** — the prompter will automatically follow your voice

### Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Space` | Play/Pause |
| `⌘,` | Open Settings |
| `⌘R` | Reset to top |

### Settings

Access settings via `⌘,` or the gear icon:

- **Script** — Edit your teleprompter text
- **Appearance** — Font, size, theme, and fade effects
- **Layout** — Window dimensions and screen position
- **Behavior** — Scroll speed and hover interactions
- **Speech** — Speech tracking, smoothing, and reading position

## Privacy

Prompter uses on-device speech recognition where possible. Your voice data is processed locally and never leaves your Mac.

Required permissions:
- **Microphone** — For speech-to-text functionality
- **Speech Recognition** — For Apple Speech framework access
